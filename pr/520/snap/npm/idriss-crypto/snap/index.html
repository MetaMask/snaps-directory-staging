Description by Web3 Address Book

Web3 Address Book can be used to resolve several naming services in MetaMask. It supports resolution of: 1. IDriss 2. Farcaster names 3. Lens handles 4. Unstoppable Domains 5. ENS domains on networks other than Ethereum Mainnet Supported formats: 1. IDriss: Registered Twitter handles @[name] 2. Farcaster: [name].fc or [name].farcaster 3. Lens: [name].lens 4. UD: any registered UD 5. ENS: Any registered ENS, given it does not resolve to a contract address.

Permissions by Web3 Address Book

  • Access the internet

  • Provide domain and address lookups
